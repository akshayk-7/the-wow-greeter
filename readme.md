# Wow Greeter CLI

A beautiful, colorful, and completely over-the top greeting command-line interface built with Node.js.

This is my very first CLI Project with Node.js.

## Features

- **Massive ASCII Banners:** powered bu `Figlet`.
- **Gradients:**Stunning retro gradient multi-color gradient using `gradient-string`.
- **Colors- Lightweight:** Uses the `picocolor` for standard terminal coloring.


## Tech Stack 

- **Node.js** (Execution Environment)
- **figlet** (ASCII text generation)
- **gradient-string** (Gradient Colors)
- **picocolors** (Standard terminal styling)

## How to run Locally?

If you want to test this CLI on your own machine, follow these steps:

1 **Clone the repository** : ``` bash  git clone https://github.com/akshayk-7/the-wow-greeter.git 
                                        cd the-wow-greeter ```

2 **Install the dependencies** : ```bash  npm install ```

3 **Link the CLI globally**: ```bash npm link ```

4. **Run the CLI** : ```bash wow-greeter ```
